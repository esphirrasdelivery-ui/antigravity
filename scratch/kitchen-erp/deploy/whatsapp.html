Como funciona

📅Todo quarta-feira, das 16h às 23h, o sistema verifica se cada loja enviou o formulário de inventário.
📱Se não enviou, uma mensagem é enviada pelo WhatsApp conectado aqui para o responsável configurado em Configurar Estoque Máximo.
🔁O lembrete é repetido a cada hora até o formulário ser enviado.
💡O servidor local precisa estar rodando no PC para os lembretes funcionarem.
